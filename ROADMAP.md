# Roadmap

This is the public roadmap for Morphase. Priorities may change.

## v0.1.x — Stabilization

- Fix edge cases in request normalization
- Improve planner explanation quality
- Harden executor error enrichment
- Expand test coverage for core flows
- Gather feedback from early users

## v0.2.x — Plugin Maturity

- Stabilize whisper plugin for local transcription
- Stabilize markitdown plugin for document extraction
- Evaluate summarize plugin maturity
- Improve install guidance for all platforms
- Add more curated pipelines where stable

## v0.3.x — UX Improvements

- Progress indicators for long-running conversions
- Better output path collision handling
- Config file documentation and examples
- Improved wizard with more route coverage

## Future Considerations

- Web UI for the local server
- Batch/multi-file processing
- Plugin auto-detection improvements
- Windows and Linux testing and hardening
- Homebrew formula
- Docker image for the server
